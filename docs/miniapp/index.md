# 📱 小程序开发实战

> **"从零到一，用项目驱动学习"** —— 小程序开发是前端工程师的重要技能，通过实战项目掌握小程序开发的完整流程。

## 为什么学习小程序开发？

```
小程序的优势
═══════════════════════════════════════════════════════

✅ 无需安装，即用即走
✅ 跨平台（微信/支付宝/抖音/快手）
✅ 流量入口多（扫码、搜索、分享）
✅ 开发成本低于原生 App
✅ 丰富的平台能力（支付、定位、摄像头）

前端工程师的小程序技能树
═══════════════════════════════════════════════════════

├── 基础能力
│   ├── WXML / WXSS / JS 三件套
│   ├── 组件化开发
│   ├── 生命周期与页面路由
│   └── 数据绑定与事件系统
│
├── 进阶能力
│   ├── 自定义组件与插件
│   ├── 网络请求与数据缓存
│   ├── 云开发（CloudBase）
│   └── 性能优化
│
└── 实战能力
    ├── 项目架构设计
    ├── 状态管理方案
    ├── 后端接口对接
    └── 上线与运维
```

```
基础入门 → 项目实战 → 进阶优化
    ↓           ↓           ↓
  官方文档    投票助手    性能优化

  生命周期    状态管理    多端适配
```

## 技术选型

```
小程序技术栈对比
═══════════════════════════════════════════════════════

原生开发        Taro             uni-app
──────────     ──────────       ──────────
微信原生语法    React 语法        Vue 语法
性能最好        跨端（React）      跨端（Vue）
学习成本低      React 生态         Vue 生态
仅微信          多端统一           多端统一

选择建议：
• 只做微信 → 原生开发
• React 技术栈 → Taro
• Vue 技术栈 → uni-app
• 需要跨端 → Taro 或 uni-app
```

## 面试要点

> 小程序开发在前端面试中通常作为**项目经验**来考察。面试官关注的是：1）你对小程序架构的理解；2）你解决实际问题的能力；3）你的工程化思维。
