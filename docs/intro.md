# 前端面试知识库

欢迎来到前端面试知识库！这里汇集了前端开发面试中常见的题目，涵盖：

## 📚 内容分类

| 分类 | 说明 |
|------|------|
| **JavaScript 基础** | 类型系统、闭包、异步、原型链、ES6+ 特性 |
| **TypeScript** | 泛型、工具类型、类型体操 |
| **React** | Hooks、Fiber 架构、状态管理、性能优化 |
| **Vue** | 响应式系统、组合式 API、虚拟 DOM |
| **AI 应用开发** | LLM 集成、RAG、流式响应、AI SDK、AI 工具 |
| **工程化** | 构建工具、CI/CD |
| **手写题** | 防抖节流、深拷贝、Promise、EventEmitter、LRU 缓存 |

## 🎯 难度说明

- 🟢 **Easy** — 基础概念，初级开发者应掌握
- 🟡 **Medium** — 进阶知识，中高级开发者必备
- 🔴 **Hard** — 深度原理，高级开发者/架构师级别

## 🤖 AI 专题

本知识库特别包含 AI 相关面试题，涵盖：
- 前端如何集成 LLM（大语言模型）
- RAG（检索增强生成）前端实现
- 流式响应（Streaming）处理
- Vercel AI SDK / LangChain.js 使用
- Cursor / Copilot 等 AI 编程工具
