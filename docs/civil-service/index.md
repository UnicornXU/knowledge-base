# 🏛️ 公务员面试

> **"面试是公务员考试的最后一公里"** —— 笔试决定门槛，面试决定成败。

## 公务员面试概述

```
公务员面试流程
═══════════════════════════════════════════════════════

笔试通过 → 资格复审 → 面试通知 → 面试 → 体检 → 政审 → 录用

面试形式：
┌─────────────────────────────────────────────────────┐
│  结构化面试（最常见）                                  │
│  ├── 5-7 名考官                                      │
│  ├── 3-5 道题                                        │
│  ├── 答题时间 15-25 分钟                              │
│  └── 百分制，去掉最高最低取平均                        │
├─────────────────────────────────────────────────────┤
│  无领导小组讨论                                       │
│  ├── 6-12 名考生一组                                 │
│  ├── 讨论时间 30-60 分钟                              │
│  └── 考官观察评分                                     │
├─────────────────────────────────────────────────────┤
│  结构化小组（税务系统等）                              │
│  ├── 3-4 名考生一组                                  │
│  ├── 依次答题 + 互评                                  │
│  └── 答题 + 点评 + 回应                               │
└─────────────────────────────────────────────────────┘
```

## 面试题型分类

```
结构化面试题型
═══════════════════════════════════════════════════════

📋 综合分析题
├── 社会现象类
├── 政策理解类
└── 名言警句类

🎯 计划组织题
├── 调研类
├── 宣传类
├── 活动策划类
└── 会议组织类

🤝 人际关系题
├── 与领导的关系
├── 与同事的关系
└── 与群众的关系

⚡ 应急应变题
├── 工作突发事件
├── 公共危机处理
└── 群众诉求处理

💬 言语表达题
├── 情景模拟
├── 演讲
└── 串词讲故事

🧠 其他题型
├── 自我认知
├── 岗位匹配
└── 漫画题
```

## 学习路线

```
推荐学习顺序
═══════════════════════════════════════════════════════

第一阶段：理论基础
  ├── 了解面试流程和评分标准
  ├── 学习各类题型的答题框架
  └── 积累时政热点素材

第二阶段：专项突破
  ├── 综合分析题（最重要，占比最高）
  ├── 计划组织题
  ├── 人际关系题
  └── 应急应变题

第三阶段：模拟练习
  ├── 对镜练习（控制表情和肢体语言）
  ├── 录音/录像回放（发现问题）
  ├── 组队模拟（适应考场氛围）
  └── 限时训练（控制答题时间）
```

## 备考建议

> 公务员面试的核心是**考察思维能力**，而不是背诵答案。考官想看到的是你分析问题、解决问题的能力，以及你是否具备公务员的素质。建议多关注时政新闻，培养政府思维。
