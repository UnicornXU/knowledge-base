# 🎯 计划组织题

> **"计划组织题考察的是做事能力"** —— 能否把一项工作安排得井井有条，是公务员的基本素质。

## 一、题型概述

```
计划组织题的常见类型
═══════════════════════════════════════════════════════

📋 调研类
"领导让你调研 XX 情况，你会怎么做？"
重点：调研对象、调研方式、调研内容

📢 宣传类
"单位要宣传 XX 政策，你会怎么组织？"
重点：宣传对象、宣传方式、宣传内容

🎉 活动策划类
"单位要组织 XX 活动，交给你负责，你会怎么做？"
重点：活动方案、组织实施、总结反馈

🤝 会议组织类
"单位要召开 XX 会议，你会怎么安排？"
重点：会前准备、会中服务、会后落实
```

## 二、通用答题框架

```
计划组织题通用框架
═══════════════════════════════════════════════════════

第一步：明确目的和意义
├── 说明这项工作的重要性
├── 明确工作目标
└── 确定工作原则

第二步：前期准备
├── 成立工作小组
├── 制定工作方案
├── 明确分工和时间节点
└── 准备物资和资料

第三步：组织实施
├── 按照方案推进
├── 各环节的具体做法
├── 突发情况处理
└── 过程把控

第四步：总结提升
├── 整理资料和数据
├── 撰写总结报告
├── 反馈改进意见
└── 建立长效机制
```

## 三、分类详解

### 3.1 调研类

```
调研类答题框架
═══════════════════════════════════════════════════════

一、调研目的
明确为什么要调研，要解决什么问题

二、调研内容（根据题目确定）
├── 现状如何
├── 存在什么问题
├── 原因是什么
└── 有什么建议

三、调研对象
├── 相关部门（获取政策和数据）
├── 服务对象（了解需求和意见）
├── 专家学者（获取专业建议）
└── 基层单位（了解执行情况）

四、调研方式
├── 问卷调查（覆盖面广）
├── 实地走访（深入了解）
├── 座谈会（集中讨论）
├── 个别访谈（深入交流）
├── 查阅资料（政策和数据）
└── 网络调查（收集民意）

五、调研报告
├── 基本情况
├── 存在问题
├── 原因分析
└── 对策建议
```

**真题示例**：

> **题目**：领导让你调研本地区老年人居家养老的情况，你会怎么做？

**参考解析**：

```
【目的意义】
老年人居家养老是重要的民生问题，做好这次调研有助于
了解老年人的实际需求，为完善养老服务体系提供依据。

【前期准备】
1. 成立调研小组，明确分工
2. 设计调研方案，确定调研内容
3. 准备调研问卷和访谈提纲
4. 联系相关部门和社区

【调研内容】
1. 老年人基本情况：年龄、健康状况、经济条件
2. 养老需求：医疗、饮食、护理、精神慰藉
3. 现有服务：社区养老服务、上门服务
4. 存在问题：服务不足、费用过高、质量不高
5. 意见建议：希望增加哪些服务

【调研对象和方式】
1. 老年人及家属：问卷 + 入户走访
2. 社区工作人员：座谈会
3. 养老服务机构：实地考察
4. 民政部门：查阅资料 + 座谈
5. 网络平台：收集网民意见

【形成报告】
整理数据，撰写调研报告，提出完善建议，
向领导汇报。
```

## 四、答题技巧

```
计划组织题高分技巧
═══════════════════════════════════════════════════════

✅ 加分项：
• 条理清晰，按时间顺序或工作步骤展开
• 细节具体，不是泛泛而谈
• 结合题目场景，有针对性
• 考虑全面，不遗漏重要环节
• 有创新点，不只是套路

❌ 扣分项：
• 流水账式，缺乏重点
• 模板化，不结合题目
• 细节缺失，过于笼统
• 遗漏重要环节
• 没有总结反馈

💡 增加亮点的方法：
• 结合数字化手段（大数据、互联网+）
• 考虑特殊群体（老年人、残疾人）
• 注重长效机制
• 强调效果评估
```

## 五、常见错误

```
考生常犯的错误
═══════════════════════════════════════════════════════

❌ 错误1：开头说太多意义
"开展这次调研非常重要，体现了领导对我的信任..."
→ 应该简洁明了，直接进入正题

❌ 错误2：方案太笼统
"我会制定一个详细的方案..."
→ 应该说清楚方案包含哪些内容

❌ 错误3：忽视总结
答完组织实施就结束
→ 一定要有总结反馈环节

❌ 错误4：不结合题目
用通用模板回答所有题目
→ 要根据具体题目调整内容

❌ 错误5：主次不分
每个环节平均用力
→ 重点环节要详细展开
```
