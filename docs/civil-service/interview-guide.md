# 👔 面试流程与礼仪

> **"第一印象决定 50% 的成败"** —— 得体的着装和礼仪能让你在面试中占据优势。

## 一、面试当天流程

```
面试当天时间线
═══════════════════════════════════════════════════════

07:00  起床，检查证件和材料
07:30  早餐（清淡为主，避免肠胃不适）
08:00  出发前往考场
08:30  到达考场，签到、抽签
09:00  候考（可能等待 1-3 小时）
       ├── 可以看资料（部分地区允许）
       ├── 不能使用手机
       └── 保持安静，调整心态
XX:XX  引导员带入考场
       ├── 敲门进入（2-3 下）
       ├── 问好："各位考官好，我是 X 号考生"
       ├── 听指令坐下
       └── 答题开始
XX:XX  答题结束
       ├── 起立
       ├── "谢谢各位考官"
       └── 离开考场（不带走任何材料）
```

## 三、考场礼仪

### 3.1 进场礼仪

```
进场流程
═══════════════════════════════════════════════════════

1. 敲门
   • 敲 2-3 下，节奏适中
   • 等待回应后进入
   • 如果门开着，也要敲门示意

2. 进门
   • 走路稳健，步伐适中
   • 微笑，目光扫视各位考官
   • 走到考生位置旁站定

3. 问好
   • 站直，双手自然下垂
   • 鞠躬 15-30 度
   • "各位考官好，我是 X 号考生"
   • 声音清晰，语速适中

4. 入座
   • 等考官说"请坐"后再坐
   • 坐姿端正，只坐椅子的 2/3
   • 双手放在桌上或膝上
   • 不要靠在椅背上
```

### 3.2 答题礼仪

```
答题时的礼仪要点
═══════════════════════════════════════════════════════

✅ 正确做法：
• 目光交流：看主考官，兼顾其他考官
• 表情自然：微笑、点头示意
• 手势适度：适当用手势辅助表达
• 语速适中：不要太快也不要太慢
• 声音洪亮：确保每位考官都能听清
• 条理清晰：使用"第一、第二、第三"

❌ 避免行为：
• 看天花板或地面（显得不自信）
• 摸头发、抖腿（显得紧张）
• 说"嗯"、"啊"等口头禅
• 打断考官的题目说明
• 答题超时
• 答完后说"我答完了"（等计时员提醒）
```

### 3.3 离场礼仪

```
离场流程
═══════════════════════════════════════════════════════

1. 答题结束
   • 等考官说"答题结束"或计时员提醒
   • 不要慌张，保持镇定

2. 起立致谢
   • 站起来，微微鞠躬
   • "谢谢各位考官"
   • 声音清晰，态度诚恳

3. 离开考场
   • 将椅子轻轻推回原位
   • 走路稳健，不要跑
   • 轻轻关门
   • 离开考场区域后再放松
```

## 四、候考技巧

```
候考时的注意事项
═══════════════════════════════════════════════════════

心理调节：
• 深呼吸放松（4-7-8 呼吸法）
• 积极心理暗示："我准备好了"
• 不要和其他考生讨论题目
• 想象自己成功答题的场景

身体调节：
• 适当活动手指和脚趾
• 按摩太阳穴放松
• 喝水润喉（不要喝太多）
• 去洗手间整理仪容

准备事项：
• 再次检查证件
• 回顾答题框架（不要背答案）
• 准备好纸笔（部分考场提供）
```

## 五、常见问题解答

**Q1: 面试紧张怎么办？**

A: 紧张是正常的，适度紧张反而能提高专注力。缓解方法：1）深呼吸；2）提前模拟练习；3）把注意力集中在题目上而不是考官身上；4）接受紧张，不要对抗它。

**Q2: 答题时可以看稿子吗？**

A: 结构化面试一般不允许看稿子，但可以在听题时简单记录关键词。无领导小组讨论可以看自己的笔记。具体以考场规则为准。

**Q3: 面试可以戴手表吗？**

A: 可以戴，但不要频繁看时间。建议戴简约款式的表，不要智能手表（可能被视为违规）。考场一般会有计时器。
