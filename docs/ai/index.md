## 🎯 学习路线

```
AI 工具（了解） → Codex 使用指南 → LLM 集成 → 流式响应 → AI SDK → RAG 实现 → 向量数据库
   (入门)           (上手)        (核心)      (重点)     (实用)    (进阶)      (高级)
```

## 面试考察重点

- **LLM 集成**：API 设计、Token 管理、成本控制
- **流式响应**：SSE vs WebSocket、前端渲染优化
- **RAG**：检索增强生成的架构设计、Embedding 策略
- **AI SDK**：Vercel AI SDK 的 useChat/useCompletion
- **实际场景**：如何构建一个 AI 聊天应用、如何实现代码补全
