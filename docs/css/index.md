# CSS 面试知识体系

CSS 是前端面试的基础考察重点之一。本模块从基础到进阶，系统梳理 CSS 核心知识点，帮助你在面试中从容应对各类问题。

## 知识模块一览

| 序号 | 主题 | 难度 | 核心内容 |
|------|------|------|----------|
| 1 | [盒模型与选择器](./box-model.md) | 简单 | box-sizing、margin 折叠、选择器优先级、继承 |
| 2 | [定位与层叠上下文](./positioning.md) | 中等 | position 五种取值、层叠上下文、z-index 规则、BFC |

| 5 | [过渡与动画](./animation.md) | 中等 | transition、@keyframes、硬件加速、性能优化 |
| 6 | [现代 CSS 新特性](./modern-css.md) | 中等 | CSS Nesting、:has()、容器查询、@layer 等 |

## 学习建议

1. **先打基础**：盒模型和选择器是所有 CSS 问题的根基，务必扎实掌握。
2. **理解布局**：Flexbox 和 Grid 是现代布局的核心，面试中出现频率极高。
3. **关注实战**：响应式设计和动画在实际项目中无处不在，结合代码练习。
4. **跟进新特性**：现代 CSS 特性是加分项，展示你对技术前沿的关注。

## 面试技巧

- 回答 CSS 问题时，**先说原理，再写代码**，最后提实际场景。
- 遇到布局问题，先分析需求再选择方案（Flex vs Grid vs 定位）。
- 提到性能优化时，结合浏览器渲染流程（重排、重绘、合成）来说明。
