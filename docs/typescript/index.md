# TypeScript

TypeScript 是现代前端开发的标配，类型系统是其核心竞争力。

## 📚 内容导航

| 文档 | 难度 | 说明 |
|------|------|------|
| [泛型](./type-generics.md) | 🟡 中 | 泛型函数、泛型约束、内置工具类型 |
| [工具类型](./utility-types.md) | 🟡 中 | Partial、Pick、Omit、Record 等 |
| [类型体操](./type-challenges.md) | 🔴 高 | 条件类型、映射类型、infer、递归 |

## 🎯 学习路线

```
泛型基础 → 工具类型 → 类型体操
 (基础)     (实用)     (进阶)
```

## 面试考察重点

- **泛型**：如何编写可复用的类型安全函数
- **工具类型**：Partial/Pick/Omit/Record 的实现原理
- **类型体操**：条件类型、`infer`、模板字面量类型
- **实际应用**：如何为 API 响应定义类型、如何约束组件 Props
