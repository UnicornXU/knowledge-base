# JavaScript 基础

JavaScript 是前端开发的根基，掌握其核心概念是面试的第一步。

## 📚 内容导航

| 文档 | 难度 | 说明 |

| [闭包与作用域](./closure-scope.md) | 🟡 中 | 闭包原理、作用域链、变量提升 |

## 🎯 学习路线

```
类型系统 → 闭包与作用域 → 原型链 → 异步编程 → ES6+ 特性
  (基础)      (核心)       (进阶)    (重点)     (加分项)
```

## 面试考察重点

- **类型判断**：`typeof`、`instanceof`、`Object.prototype.toString` 的区别
- **闭包应用**：防抖、节流、模块化、柯里化
- **事件循环**：微任务与宏任务的执行顺序
- **原型链**：继承方式、`new` 的实现原理
- **ES6+**：箭头函数、Promise、Proxy、Iterator
